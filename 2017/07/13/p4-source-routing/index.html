P4 Exercise -- Source Routing

July 13, 2017 Tutorial

这个demo是在SIGCOMM 2015 P4 Tutorial的一个练习。是让用户使用P4实现一个非常简单的称为EasyRoute的源路由转发协议。本文假设您已经完成P4软件交换机的安装。

代码仓库：https://github.com/p4lang/tutorials/tree/master/SIGCOMM_2015/source_routing

EasyRoute的数据包结构如下：

各字段的意义如下：

  • preamble: 没有实际意义，主要用于填充，估计还能和后面num_valid字段一起组成12Bytes，让网卡以为是dst和src MAC地址。
  • num_valid: 表示数据包的剩余的跳数。
  • port_X: 表示交换机出口的编号，在每一跳上只有最外层的port字段有效。

其仓库README中有对source_routing运行原理详细的解释。
